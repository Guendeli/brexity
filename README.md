# Brexity 2031

Brexity 2031 is a game in which you'll control an immigrant trapped at London in 2031.

This is a stealth/adventure game.

## Quickstart

1. Install [Unity](http://unity3d.com/)
2. Open the project
3. Build with Unity or Visual Studio
4. Enjoy \o/
